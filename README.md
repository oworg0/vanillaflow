# ✦ Lumina

> Lumina Modpack is a feature-rich Fabric modpack focused on visual enhancement, smooth performance, and immersive atmosphere — featuring shaders, dynamic lighting, ambient sounds, and a polished UI.

<details>
<summary>🌅 About</summary>

## 🌅 Visual
- Particles, visual effects & smooth transitions
- 3D skin layers & detailed armor bar
- Pack shaders and dynamic lighting
## 🔊 Audio
- Ambient sounds for every biome and dimension
- Presence Footsteps — unique sounds for every surface
- Simple Voice Chat support
## ⚙️ Quality of Life
- Minimap and world map based on loaded chunks
- Smooth mouse wheel control
- Improved game statistics, saturation bar display and detailed melting information
- A clear screen. A more transparent F3, dynamic crosshair, and convenient, smooth screen zoom
## ⚡ Performance
- Sodium + Iris + Krypton + FerriteCore
- Optimized for mid-range hardware

</details>

<summary>⚙️ Use modpack</summary>

I(ш) - turn on Do a Barrel Roll

G(п) - increase brightness

B(и) - EmoteCraft's wheel of emotions

X(ч) - turn on/off the Simple Voice Chat microphone

N(т) - disable Simple Voice Chat

Z(я) - enlarge minimap

M(ь) - open world map

C(с) - zoom

K(л) - enable/disable selected shaders in Iris

O(щ) - list of available shaders

F4 - freecam

](ъ) - settings Xaero's World Map

Y(н) - settings Xaero's Minimap

V(м) - settings Simple Voice Chat

! All other mods are configured in the same way in ModMenu !

</details>
